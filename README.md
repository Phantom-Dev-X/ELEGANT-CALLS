## 🎯 About

**ÉLÉGANT CALLS** is a luxury surprise gifting service based in Lagos, Nigeria. We deliver personalized phone calls, curated gift packages, pastries, and accessories — all wrapped in elegance and emotion.

This is the **landing page** for the service, built from scratch as a **learning project** to practice real-world frontend development.

> 🌱 **I'm a beginner developer** actively learning HTML, CSS, Tailwind CSS, and JavaScript. This project is my way of applying what I learn — one commit at a time. It's not perfect yet, but every push makes it better. I hope to secure a frontend developer role soon and keep growing!

---

## 📐 Design Decisions

| Decision | Why |
|----------|-----|
| **Mobile-first** | Most Nigerian users browse on phones |
| **Tailwind CDN** | No build tools needed, beginner-friendly setup |
| **Base64 images** | Self-contained single file, loads on any host |
| **JS hamburger menu** | More intuitive than CSS checkbox hack |
| **`clamp()` for sizing** | Smooth responsive scaling without breakpoints |
| **Single HTML file** | Simplicity while learning — will refactor later |

## 🚀 Getting Started

1. **Clone the repo**
   ```bash
   git clone https://github.com/Phantom-Dev-X/ELEGANT-CALLS.git
   cd ELEGANT-CALLS
   ```

2. **Open in browser**
   ```bash
   # Just open index.html — no build step needed!
   open index.html
   ```

That's it. No `npm install`, no build tools, no server needed. Just HTML, CSS, and JS. ✅
